 Voice Assistant 🎙️

A simple Python-based voice assistant that listens to your voice, responds with speech, and performs tasks like opening websites, fetching info from Wikipedia, telling jokes, and more.

Features

- 🎤 Voice input using your microphone
- 📢 Text-to-speech responses
- 🌐 Opens YouTube and Google
- 🧠 Wikipedia search
- ⏰ Tells current time and date
- 😂 Tells random jokes
- ❌ Graceful exit with "bye" or "quit"

Requirements

Install dependencies using:

```bash
pip install -r requirements.txt
